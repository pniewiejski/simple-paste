## ymmm... but why?

Why does this repo exist? Well, I'm glad you asked handsome stranger! To tell you the truth, since
the whole `#stayhome` thing had started, I had some spare time to catch up with a few books &
articles I wanted to read for some time. When I was going through the awesome
[system-design-primer](https://github.com/donnemartin/system-design-primer) article, I found this
system's design question -
[Design Pastebin.com](https://github.com/donnemartin/system-design-primer/blob/master/solutions/system_design/pastebin/README.md).
